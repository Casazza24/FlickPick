# Wireframes — FlickPick Team 1302

Low-fidelity wireframes for each page of the FlickPick application. Replace each placeholder file below with your actual wireframe image.

---

## Required Files

| File | Page | Key Elements to Show |
|------|------|----------------------|
| `login.png` | Login | Username field, password field, sign in button, link to register |
| `register.png` | Registration | Username, email, password fields, create account button, link to login |
| `home.png` | Home / Swipe | Movie card (poster, title, genre, rating, synopsis), Pass and Save buttons, nav bar |
| `watchlist.png` | Watchlist | Grid of saved movie cards, remove button per card, nav bar |
| `group_session.png` | Group Watch Session | Create/join session input, participant list, mutual match results |
| `profile.png` | User Profile | Username, favorite genres, swipe history stats, logout button |

---

## Guidelines

- Wireframes should be **low-fidelity** — boxes, labels, and layout only. No colors or final styling needed.
- Show **navigation elements** (header/nav bar) on every page wireframe.
- Hand-drawn sketches photographed as images are acceptable.
- Recommended tools: [Figma](https://figma.com), [draw.io](https://draw.io), Balsamiq, or pen and paper.
- Export or save as `.png` and **replace the placeholder file of the same name** in this directory.
